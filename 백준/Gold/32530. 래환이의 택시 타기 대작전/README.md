### 문제 설명

<p>한과영 학생들은 귀가할 때 택시를 타고 부산역에 간다. 아쉽게도 캐리어 크기 때문에 하나의 택시에는 최대 $3$명의 학생만이 탑승할 수 있다. 또한 본인이 원하는 귀가 시각에서 $10$분 전후 내로 택시를 탑승해야 한다. 단, 학생들은 모두 같은 날에 택시를 타야 한다.</p>

<p>하지만 귀갓날에 래환이는 벌칙에서 지는 바람에 $N$명의 학생들의 택시비를 내기로 했고, 돈이 없는 래환이는 필요한 택시의 대수를 최소화하고 싶었다. 학생별로 본인이 원하는 귀가 시각이 주어질 때 필요한 택시의 최소 대수를 구해 래환이를 도와주자!</p>

### 입력 

 <p>첫 번째 줄에는 정수 $N$$(1 \le N \le 2 \times 10^5)$이 주어진다.</p>

<p>다음 $N$개의 줄 중 $i$번째 줄에 $i$번째 학생이 원하는 귀가 시각이 주어진다. 시각은 <span style="color:#e74c3c;"><code>HH:MM</code></span> 형태로 주어지고, <span style="color:#e74c3c;"><code>00:00</code></span>부터 <span style="color:#e74c3c;"><code>23:59</code></span>까지로 표현된다.</p>

### 출력 

 <p>학생들이 모두 택시를 타고 귀가하기 위해 필요한 택시의 최소 대수를 출력한다.</p>
